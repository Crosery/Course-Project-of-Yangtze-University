# 长江大学WEB课设--柚子厨蒸鹅心

本项目是一个基于Web技术的静态网站，项目采用原生的Web技术栈实现，展现了丰富的多媒体内容和交互效果。

## 项目结构

```
├── index.html          # 主页面
├── html/              # HTML页面目录
│   ├── index2.html    # 二级页面
│   ├── start.html     # 开始页面
│   └── js.html        # JavaScript演示页面
├── img/               # 图片资源目录（包含柚子厨相关图片）
├── js/                # JavaScript脚本目录
├── music/             # 音频资源目录
└── video/             # 视频资源目录
```

## 技术栈
- HTML5
- CSS3
- JavaScript

## 主要功能
- 柚子厨文化展示
- 鹅心主题内容
- 多媒体资源播放
- 响应式页面设计
